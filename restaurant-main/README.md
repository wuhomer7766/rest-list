

<h2>功能描述</h2>

<p>這是一個尋找餐廳的網頁，您可以：</p>
<p>1.點選感興趣的餐廳，查看該餐廳的詳情</p>
<p>2.在搜尋欄內輸入您需要查找的餐廳名稱或分類，即可篩選出您要的餐廳</p>


************************************************************************




<h2>安裝與執行步驟</h2>

打開終端機，複製一份專案至您的電腦，在終端機中輸入 
<pre>git clone https://github.com/Yanhuabcd820/restaurant</pre>


安裝express，在終端機中輸入 
<pre>npm install express</pre>


安裝nodemon，在終端機中輸入 
<pre>npm install -g nodemon</pre>
